# NexRoute AI – Smart Route Planner

## Overview
NexRoute AI is a Streamlit-based intelligent routing system that optimizes delivery routes by balancing cost, delivery time, and environmental impact.

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Start the application:
   streamlit run app.py

## Key Features
- Multi-objective route optimization
- Interactive weight-based decision making
- Cost, time, and CO₂ trade-off visualization
- Downloadable optimization results

## Tech Stack
Python, Streamlit, Pandas, Plotly
